Dispatch

The dispatch plugin can help you run build job in a separate window. If you are using tmux, it will create the window in tmux.

If the build job contains any error, it will prompt up a quick fix list.

Usage

You can use command:

:Dispatch <Your Commands>

or keymap ;d to open the command prompt.

keymaps

keymapsfunction
m<CR>:Make<CR>
m<Space>:Make<Space>
m!:Make!
m?Show 'makeprg'
`<CR>:Dispatch<CR>
`<Space>:Dispatch<Space>
`!:Dispatch!
`?:FocusDispatch<CR>
'<CR>:Start<CR>
'<Space>:Start<Space>
'!:Start!
'?show b:start
g'<CR>:Spawn<CR>
g'<Space>:Spawn<Space>
g'!:Spawn!
g'?Show 'shell'
