Nvim Guide Book

Keymap

Leader key

leader key is set to Space.

Navigation

Basic movement

  ^
  k
<h l>
  j
KeymapFunction
HGo to the first character of the line
LGo to the last character of the line
JGo down 5 lines
KGo up 5 lines
WGo 5 words forward
BGo 5 words backward

Cursor move in insert mode

KeymapFunction
Ctrl hAct like Home
Ctrl eAct like End
Ctrl fAct like w in normal mode
Ctrl bAct like b in normal mode

Windows Navigation

KeymapFunction
;kGo to the window above
;jGo to the window below
;hGo to the right window
;lGo to the left window

Windows resize

KeymapFunction
Ctrl Shift UpResize the top border of window
Ctrl Shift DownResize the bottom border of the window
Ctrl Shift LeftResize the left border of the window
Ctrl Shift RightResize the right border of the window

Search

KeymapFunction
NGo to the before matches
nGo to the following matches
ESCClose the search highlight

Tabpages

KeymapFunction
Ctrl-t hGo to the previous tab
Ctrl-t lGo to the next tab
Ctrl-t nCreate new tab

Copy and Paste

KeymapFunctionnotes
yCopy to register
pPaste from the register
Ctrl-pPaste from the clipboardIn normal and insert mode
Ctrl-yCopy to clipboardOnly in visual mode

Text move

KeymapFunctionnotes
<Reduce one indent levelIn normal and selection mode
>Increse one indent levelIn normal and selection mode

Save and Quit

KeymapFunctionNotes
; wSave
; qQuit buffer (Auto quit nvim when last buffer is deleted)
: qQuit neovim, window, tabs
Alt ;Leave the insert mode (Same as ESC)Only in insert mode

Others

KeymapFunction
Ctrl-zReverse changes
